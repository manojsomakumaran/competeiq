### Added
- Initial production-grade package scaffold
- Dual-environment (local + Colab) configuration layer
- Multi-agent orchestration: Price, Catalog, Marketing

- Product knowledge graph (NetworkX)
- Gradio UI with 4 tabs (Category Analysis, Product Search, Price Comparison, System Status)

- Extended sample datasets: Companies X, Y, Z, W across 6 categories
- Unit, IT-API, and IT-UI test suites
- Docker + AWS ECS/Fargate deployment assets
- GitHub Actions + GitLab CI pipelines
